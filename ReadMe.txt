Sujet choisi : Application de gestion des stocks d'un magasin de vélos

Comment lancer le projet :

--- Éxécuter le fichier src/main/Main.java ---

S'affiche alors dans le terminal le visuel suivant :
_______________________________________________

Voici la liste des actions :

Commandes : 

[1] Consulter une commande
[2] Consulter toutes les commandes
[3] Créer une commande
[4] Exporter les commandes
[5] Supprimer une commande


Clients : 

[6] Consulter un client
[7] Consulter tous les clients
[8] Créer un client
[9] Exporter les clients
[10] Supprimer un client


Articles : 

[11] Consulter un article
[12] Consulter tous les articles
[13] Créer un article
[14] Exporter les articles
[15] Supprimer un article
_______________________________________________

Un export crée un fichier csv avec les informations relatives au client/commande/article exporté

Une consultation affiche les les informations relatives au client/commande/article sur la fenêtre du terminal

Une création crée un objet client/commande/article dans la base de donnée

Une suppression supprime un objet client/commande/article de la base de donnée
